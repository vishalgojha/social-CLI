# Social Flow Domain Skills

These skills keep the agent beginner-friendly while still powerful.

## Available Skills

- `setup-auth.md` - onboarding, token setup, app credentials, health checks.
- `facebook.md` - Pages, profile checks, post workflows, read patterns.
- `instagram.md` - IG account/media flows, publish/read guardrails.
- `waba.md` - WhatsApp Cloud API, templates, webhooks, send safety.
- `marketing.md` - ad accounts, campaign checks, spend-aware guidance.

## How Hatch Uses Them

- Hatch auto-detects domain intent and routes to the best skill.
- If intent is unclear, Hatch still proposes domain-specific next actions.
- Detailed diagnostics remain available with `social hatch --verbose`.
