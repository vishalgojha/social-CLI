# Skill: WABA (WhatsApp Business)

## Goal

Make WhatsApp Cloud API tasks safe and understandable for beginners.

## Core Tasks

- phone number ID checks
- template send guidance
- webhook setup validation
- integration sanity checks

## Conversation Rules

- Treat phone IDs, template names, and recipients as separate fields.
- Explain where each value comes from before prompting.
- Offer copy-ready example commands.

## Guardrails

- Validate recipient formatting before send.
- Require confirmation for message sends.
- For webhook issues, provide deterministic fix checklist.
